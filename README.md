In this project concepts used are
<ul>
  <li>OPPS</li>
  <li>File handling</li>
  <li>Linked List</li>

  <li>insertion sorting (also implemented concept of order by as in sql i.e. order by partyname,  state, city)</li>
  <li>index mapping</li>

</ul>
